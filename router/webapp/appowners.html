Admin console

First time user?

  • TO REQUEST ACCESS email DL ApplicationServices Admin

New Application

  • Open the admin console here and select Manage Applications
  • Click on New Application
  • Fill in App Name and Save
  • For adding to Jamf complete the help form.

Enter Items

  • Click Edit on your application and add Contacts, Keywords, Releases, Help or Announcements
  • To add a new IPA, you need to create a new Release. Once the release is saved you will be able to upload to Jamf.

Support Center

If using the iOS feedback service. Feedback is inserted into the backend and can be viewed and responded to via the Fiori Feedback Admin Tool.

If using ASC SDK see below for setting up the link for feedback

  • Click on the edit on your application
  • Scroll down to the Support and Feedback Section
  • Select from Feedback dropdown
    • Email - enter email address - best to use a DL List

      If using a Distribution List, in your email right click on as if viewing the members, then click on the envelope. This will give you the real outlook name for your support email. It will look something like DL_548F39BCFD84A03077000005@exchange.sap.corp.

      Note: If you leave the email blank then the "Send Feedback" link will NOT show.

    • Ticket Category
      • For ITDirect enter something like IMIS_PEQP_MOBIL (category for mobile apps)
      • For Unified Ticketing enter something like ZINE/SRIS_APPL_ADVICE or ZGCR/null.
        You must have a "/" to separate the process type and category.
    • Email/Ticket Body - will display body in email or in ticket description
      • In an email this is the body of the email
      • In a ITDirect or Unified ticket this is the subject and description
