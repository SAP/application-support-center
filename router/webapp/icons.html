First time user? Need help with icon?

Icon Center

  • Go to APP ICON Jam request page.
  • Enter:
    • Name of Application
    • Description of Application
    • Category
    • (Optional) An image you would like the icon design to follow. We design our image on a Fiori-inspired style. You can take a look at https://openui5.hana.ondemand.com/iconExplorer.html for icon ideas.
    • (Optional) When you would like the icon by
    • Any other information you would like us to know (e.g. if you would also like icons for Apple Watch compatibility)
